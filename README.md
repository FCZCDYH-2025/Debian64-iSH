# Debian64-iSH
It can be used in iSH Shell (AOK).But it must be the latest iSH-AOK.Because it is AMD64 version
## Do not use in business,this is only for exploratory research!
# How to use it?
First,install iSH-AOK in Testflight.You can also try another emulator like proot,chroot or qemu.
Next,download rootfs file in release.The rootfs apt list is Tsinghua's list.When you are in another country.Please set the source.list to Debian.org!You must first configure the iSH to Plain Terminal and then boot the filesystem.
Finally,you can start the debian travelling.
